★ Task 10: Quản lý Mã Giảm Giá (Coupon System) TASK MỚI ★

Tính năng Marketing thực tế nhà hàng. Thầy giáo đánh giá cao nếu Có.

  • Bước 1 - Tạo Entity mới Coupon.cs: 
    public class Coupon {
    public int Id { get; set; }
    [Required] public string Code { get; set; }    // VD: "GIAM20K"
    public string DiscountType { get; set; }        // "percent" hoặc "fixed"
    [Range(0, 10000000)] public decimal DiscountValue { get; set; }  // 20 (%) hoặc 50000 (đ)
    public decimal? MinOrderAmount { get; set; }    // Đơn tối thiểu
    public int MaxUsage { get; set; }               // Số lần dùng tối đa
    public int UsedCount { get; set; } = 0;
    public DateTime ExpiryDate { get; set; }
    public bool IsActive { get; set; } = true;
}
  • Bước 2 - Tạo CouponsController.cs:
    - [HttpGet] Lấy danh sách (Admin).
    - [HttpPost] Tạo mã mới (Admin).
    - [HttpPut] Sửa mã (Admin).
    - [HttpDelete] Tắt mã (Soft Delete → IsActive = false).
    - [HttpPost("validate")] Khách gửi lên mã + tổng tiền → API kiểm tra hợp lệ → Trả về số tiền được giảm.
  • Bước 3 - Validate mã (Business Rules):
    - Mã không tồn tại → "Mã giảm giá không hợp lệ"
    - Mã hết hạn (ExpiryDate < Now) → "Mã đã hết hạn"
    - Mã dùng hết lượt (UsedCount >= MaxUsage) → "Mã đã hết lượt sử dụng"
    - Đơn chưa đủ tối thiểu → "Đơn hàng phải từ X đồng trở lên"
    - ✅ Hợp lệ → Trả giá trị giảm, Frontend hiện: "Giảm 20% (-60.000đ)"
  • Bước 4 - UI Admin: Trang Admin/Coupons.cshtmlbảng quản lý (Tạo/Sửa/Ẩn mã).
  • Bước 5 - UI Khách: Trong Cart.cshtml, thêm ô "Nhập mã giảm giá" + nút "Áp dụng". Khi hợp lệ, hiển thị dòng trừ tiền trên bill.
  • Bước 6 - Khi thanh toán: Lưu CouponId vào Order, tăng UsedCount lên 1. Thêm cột DiscountAmount vào Entity Order để lưu số tiền đã giảm.
